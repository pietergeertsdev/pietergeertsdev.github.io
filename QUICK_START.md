# Quick Start Guide - Oak Heritage Home Website

## 🚀 Immediate Next Steps

### 1. View the Website
The website should now be open in your browser. If not:
- Double-click `index.html` in Finder
- Or right-click → Open With → Your preferred browser

### 2. What You'll See
- **Hero Section**: Full-screen image with title and call-to-action
- **Story Section**: Information about the house's history
- **Gallery**: 10 property images in a grid layout (click to view full-size)
- **Features**: Key property highlights
- **Location**: Information about the area
- **Contact Form**: For potential buyers to reach you

### 3. Test These Features
✅ **Navigation Menu**
   - Click menu items to scroll to sections
   - On mobile: Click hamburger icon (☰) to open menu

✅ **Image Gallery**
   - Click any image to open lightbox
   - Use arrow buttons or keyboard (← →) to navigate
   - Press ESC or click X to close

✅ **Responsive Design**
   - Resize browser window to see mobile/tablet layouts
   - Test on your phone/tablet

✅ **Contact Form**
   - Fill out and submit (shows demo alert for now)

## 🔧 Common Issues & Fixes

**Images not showing?**
- Check image filenames match exactly (case-sensitive)
- Ensure images are in same folder as index.html

**Styling looks broken?**
- Verify styles.css is in same folder
- Check browser console for errors (F12)

**JavaScript not working?**
- Verify script.js is in same folder
- Check browser console for errors

**Mobile menu not working?**
- Clear browser cache
- Try different browser
